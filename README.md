# Billion Seconds

Find out how many seconds you've been alive — and when your one-billionth second falls.

Enter your birthday and the app shows your live second count, your billionth-second milestone date, and lets you share it.

## Development

```bash
npm install       # Install dependencies
npm run start     # Dev server with hot reload → http://localhost:5173
npm run build     # Production build → dist/
npm run preview   # Preview the production build locally
```

## Deploy

Pushing to `main` automatically builds and deploys to GitHub Pages via GitHub Actions.
