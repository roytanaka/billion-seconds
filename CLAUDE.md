## Architecture

Single-page vanilla JS app built with Vite 5. No framework.

**Entry point:** `src/main.js` ‚Äî wires up DOM selectors, event listeners, and a 100ms `setInterval` that updates the live second count.

**Core logic:** `src/date-builder.js` ‚Äî exports date/time option arrays, `makeBday(y, m, d, h, min)` (returns a Day.js object), and `getSecondsData(bday)` which returns milestone dates for the billionth and two-billionth seconds.

**Renderer:** `src/renderer.js` ‚Äî `initRenderer(container, onStartOver)` creates stable DOM nodes inside `#seconds`, appends a "Try another date" button wired to `onStartOver`, and returns an `update(secondsData)` function. Called once after modal submit; the interval calls `update()` on each tick.

**Sharing:** `src/share.js` ‚Äî `initShare(container, getSecondsData)` appends a Share button. On touch devices uses the native Web Share API; on desktop opens a custom popover with Copy link (clipboard API) + X/Facebook fallback links. Called once after modal submit alongside `initRenderer`.

**Starfield:** `src/stars.js` ‚Äî Canvas-based animated starfield. Exports `initStars()` and `setSpeedMultiplier(target, durationMs)`. The speed multiplier tween is built in for future use (Phase 2 warp effect on "Go" click). Three star size tiers match the original CSS layered effect.

**Styles:** `src/css/style.css` imports 8 component partials. CSS custom properties (`--dark-bg`, `--dark-bg-lighter`, `--dark-bg-hover`) defined in `:root` in `style.css`. Uses native CSS nesting. PostCSS/autoprefixer runs via `postcss.config.js`. Note: `@import` rules must come before all other statements or PostCSS will warn.

**Theme color:** `#66fcf1` (teal) is defined only in `src/stars.js` as `STAR_COLOR` ‚Äî not as a CSS custom property. Change it there, then update the hardcoded hex values in `seconds.css`, `modal.css`, and `date-picker.css`.

**Build output:** `dist/` ‚Äî Vite hashes asset filenames. Legacy bundle (`index-legacy-*.js`) generated by `@vitejs/plugin-legacy` for broad browser support.

## Key Dependencies

- `dayjs` ‚Äî all date arithmetic and formatting (replaced moment.js)
- `@vitejs/plugin-legacy` ‚Äî transpiles for older browsers via Babel under the hood
- `autoprefixer` ‚Äî vendor prefixes via PostCSS
